Utilisateurs

Cette page a pour objectif d'analyser l'évolution des utilisateurs de la plateforme Arrivage et les interactions entre ces derniers. Pour se faire, nous avons développé deux visualisation interactive que vous pouvez explorer ci-dessous.

Évolution des utilisateurs

Ce graphique a pour but de visualiser l'évolution des utilisateurs au fil du temps. Trois courbes différentes permettent de montrer l'évolution du nombre d'acheteurs, de vendeurs et de l'ensemble des utilisateurs.

Deux modes d'affichage sont disponibles: empilé et continu. Le mode continu permet de visualiser l'évolution depuis le lancement de l'application jusqu'à aujourd'hui. Le mode empilé permet de comparer l'évolution de chaque année.

Enfin, deux modes de calcul sont disponibles: cumulatif et absolu. Le mode absolu se concentre sur le nombre de nouveaux utilisateurs chaque mois, tandis que le mode cumulatif montre l'évolution totale des utilisateurs au fil du temps.

Relations vendeurs-acheteurs

Cette visualisation interactive présente un diagramme de réseau qui illustre les connexions entre vendeurs et acheteurs. Explorez de deux façons :

Vue filtrée : Choisissez un vendeur pour voir ses acheteurs, ou un acheteur pour voir ses vendeurs.
Vue globale : Observez l'ensemble du réseau commercial et filtrez en cliquant sur un nœud.

Fonctionnalités interactives : zoom et déplacement dans le graphique, survol des nœuds pour voir les détails (nombre de commandes, catégories de produits pour les vendeurs), clic sur un nœud pour filtrer le réseau et voir uniquement ses connexions.

Les nœuds bleus représentent les acheteurs, les verts les vendeurs. La taille des nœuds reflète le volume de commandes.
